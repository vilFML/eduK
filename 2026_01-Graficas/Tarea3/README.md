# Tarea 3: Rebotes en Plano

Programa que implementa colisiones entre una malla de terreno irregular con esferas que pueden rebotar de distinta forma según su materialidad.


## Requerimientos
**Se necesita instalar el módulo rtree**:
```py
pip install rtree
```


## Instrucciones
- Movimiento de cámara con `W`,`A`,`S` y `D`.
- Para elegir material de pelota: Flechas `Up` y `Down`.
- Para alternar posición de spawn: Flechas `Left` y `Right`
- Para soltar pelota: `Space`
